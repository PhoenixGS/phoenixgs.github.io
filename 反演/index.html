反演原理

设

\[ g_n = \sum_{i = 0} ^ n a_{ni}f_i \]

\[ \sum_{i = 0}^n b_{ni}g_i = f_n \]

因此，反演公式要成立，即

\[ \sum_{k = j}^ib_{ik}a_{kj} = [i = j] = \delta_{ij} \]

\[ \sum_{k = j}^ia_{ik}b_{kj} = [i = j] = \delta_{ij} \]
